# MongoDB Query Cheatsheet

## Операторы сравнения
- `$eq` — равно
- `$ne` — не равно
- `$gt` / `$gte` — больше / больше или равно
- `$lt` / `$lte` — меньше / меньше или равно
- `$in` — в массиве
- `$nin` — не в массиве

## Логические операторы
- `$and` — И
- `$or` — ИЛИ
- `$not` — НЕ
- `$nor` — ни одно из условий

## Операторы для массивов
- `$all` — содержит все элементы
- `$elemMatch` — элемент соответствует всем условиям
- `$size` — размер массива

## Агрегация (стадии)
- `$match` — фильтрация
- `$group` — группировка
- `$sort` — сортировка
- `$limit` / `$skip` — пагинация
- `$project` — проекция полей
- `$lookup` — JOIN с другой коллекцией
- `$unwind` — разворачивание массивов
