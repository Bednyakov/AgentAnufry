# Решение проблемы с поиском в Google

## Проблема
Агент не мог выполнить поиск в Google из-за блокировки автоматических запросов и требования пройти reCAPTCHA. Google легко определяет headless браузеры по характерным признакам автоматизации.

## Решение

### 1. Антидетект меры в браузере
Обновлен файл `tools/browser.py` с добавлением:

- **Отключение флага автоматизации**: `--disable-blink-features=AutomationControlled`
- **Реалистичный User-Agent**: Mozilla/5.0 с актуальной версией Chrome
- **JavaScript инъекции** для скрытия признаков автоматизации:
  - Переопределение `navigator.webdriver` → `false`
  - Добавление объекта `window.chrome`
  - Эмуляция плагинов и языков браузера
- **Имитация человеческого поведения**:
  - Задержки между действиями (1-2 секунды)
  - Постепенный ввод текста с задержкой (delay=100ms)
  - Настройка viewport, локали и часового пояса

### 2. Новый инструмент `browser_search_google`

Добавлена специализированная функция для поиска в Google:

```python
async def browser_search_google(query: str) -> Dict[str, Any]:
    """
    Выполняет поиск в Google и возвращает результаты.
    Использует антидетект меры для обхода блокировок.
    """
```

**Возвращает:**
```json
{
  "success": true,
  "query": "грузоперевозки из Москвы в Якутск",
  "results": [
    {
      "title": "Заголовок результата",
      "link": "https://example.com",
      "snippet": "Описание результата..."
    }
  ],
  "count": 10
}
```

### 3. Интеграция в агента

Инструмент добавлен в `main.py`:
- Описание для LLM в массиве `TOOLS`
- Обработчик в функции `execute_tool`
- Упоминание в `SYSTEM_PROMPT`

## Использование

### Для агента (автоматически)
Агент теперь автоматически использует `browser_search_google` вместо `browser_navigate` для поиска в Google:

```
💬 Вы: Найди информацию о грузоперевозках из Москвы в Якутск

🔧 Вызов: browser_search_google({"query": "грузоперевозки из Москвы в Якутск"})
📤 Результат: {"success": true, "results": [...], "count": 10}
```

### Программно
```python
from tools.browser import browser_search_google

result = await browser_search_google("ваш запрос")
if result["success"]:
    for item in result["results"]:
        print(f"{item['title']}: {item['link']}")
```

## Дополнительные рекомендации

### 1. Режим браузера
По умолчанию браузер запускается в **видимом режиме** (`headless=False`). Это помогает обойти некоторые проверки Google.

Для переключения в headless режим измените в `tools/browser.py`:
```python
_browser = await _playwright.chromium.launch(
    headless=True,  # Изменить на True
    args=[...]
)
```

### 2. Альтернативные поисковые системы
Если Google продолжает блокировать запросы, рассмотрите использование:
- **DuckDuckGo** - не требует обхода защиты
- **Bing** - менее строгие проверки
- **Yandex** - хорошо работает для русскоязычных запросов

### 3. Использование прокси
Для более надежной работы можно добавить ротацию прокси:
```python
context = await _browser.new_context(
    proxy={
        "server": "http://proxy-server:port",
        "username": "user",
        "password": "pass"
    }
)
```

### 4. Задержки между запросами
Не делайте слишком много запросов подряд. Добавьте задержки:
```python
await asyncio.sleep(random.uniform(3, 7))  # 3-7 секунд между запросами
```

### 5. Cookies и сессии
Для долгосрочного использования сохраняйте cookies:
```python
# Сохранение
await context.storage_state(path="google_state.json")

# Загрузка
context = await _browser.new_context(storage_state="google_state.json")
```

## Возможные проблемы

### Google все еще показывает CAPTCHA
1. Убедитесь, что браузер запущен в видимом режиме
2. Попробуйте использовать прокси
3. Увеличьте задержки между действиями
4. Используйте альтернативные поисковые системы

### Не находятся элементы на странице
Google часто меняет структуру HTML. Обновите селекторы в функции `browser_search_google`:
```python
search_results = await page.query_selector_all('div.g, div[data-sokoban-container]')
```

### Браузер не запускается
Убедитесь, что Playwright установлен с браузерами:
```bash
pip install playwright
playwright install chromium
```
